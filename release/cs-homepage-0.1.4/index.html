About

I'm an undergraduate at McGill University, where I am studying computer science with minors in political theory and the history and philosophy of science. My other academic interests include political science, Canadian history, and linguistics. In my free time I like to play soccer and guitar.

Projects

Future Explorer

A single–page web application for career planning and exploration. The goal will be to aggregate useful information released by Canadian governments along with some tools to help users apply it to their plans for the future. This is a publicly accessible alpha release.

European Territories in North America

An interactive map of the European-dominated territories in North America since 1713. It will be possible to click on each region and see its development over the centuries. The map is rendered using inline SVG.

“Sorry!” A desktop implementation

This emulation of the board game “Sorry!” was one of my earlier projects. It supported both AI and multiple human players as well as multi-turn undo/redo. It was written in Python using Tkinter for the user interface.

(As the game is still manufactured by Hasbro, I have not released any code from this project publicly.)
